VERSION 1:
En esta versión, el software procesa una lista de aeropuertos y permite modificarla ya sea añadiendo o eliminando aeropuertos de dicha lista, procesando a la vez un fichero KML para su visualización en Google Earth con un código de colores que permite distingir entre aeropuertos pertenecientes o no a la zona Schengen (verde en el caso de que pertenezcan y rojo en el que no). La interfaz nos permite ver las coordenadas de cada aeropuerto y si pertenecen o no a la zona Schengen. 

Version 2: en esta versión añadimos las funciones para el trabajo con el fichero arrivals, a partir del cual establecemos gráficos sobre aerolíneas y tipos de vuelo operando desde LEBL y añadiendo también otro fichero KML para su visualizacion en Google Earth sobre las rutas operadad con origen Barcelona.
